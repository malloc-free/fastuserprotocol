tb_session_t *  data

Add work to a worker.

Add work to the given worker. Is thread-safe, so it will block if another thread is adding work to the queue. The work is added to the work queue in the given worker.

Precondition
worker must be of struct type tb_worker_t.
data must be of type tb_session_t. parameters can be NULL.
Parameters

void *  value

Compare two workers, required by gdsl_heap.

gdsl_heap_t is a max heap, so workers with a lesser number of tasks will return > 0, same number of tasks = 0, and greater number of tasks < 0.

Precondition
E and value must be both of struct type worker.
E and value cannot be NULL.
Parameters
EThe element to compare.
valueThe element to compare against.
Returns
-1, 0 and 1 if E is greater, equal, or less than.
